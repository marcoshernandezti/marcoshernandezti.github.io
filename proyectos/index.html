Proyectos

A continuación se presentan algunos proyectos representativos enfocados en desarrollo backend, análisis técnico y modernización de sistemas. Estos proyectos reflejan mi forma de abordar problemas reales desde una perspectiva técnica y estructurada.

Modernización backend en .NET 8

Proyecto orientado a la construcción de una solución backend moderna utilizando .NET 8, aplicando arquitectura en capas y buenas prácticas de desarrollo.

Aspectos destacados:

  • Separación clara de responsabilidades
  • Uso de Entity Framework Core
  • Persistencia mediante procedimientos almacenados
  • Enfoque en mantenibilidad y escalabilidad
Ver repositorio en GitHub

Buscador con paginación y análisis de datos

Implementación de un buscador con paginación a partir de información estructurada en XML, diseñado como ejercicio de análisis, transformación de datos y manejo eficiente de resultados.

Aspectos destacados:

  • Procesamiento y consumo de datos estructurados
  • Lógica de paginación orientada a rendimiento
  • Base para evolución hacia arquitectura en capas
Ver repositorio en GitHub

Optimización y análisis de bases de datos

Diferentes iniciativas orientadas al análisis y optimización de consultas SQL, procedimientos almacenados e índices, con foco en mejorar tiempos de respuesta y estabilidad de sistemas existentes.

Aspectos destacados:

  • Análisis de rendimiento en sistemas productivos
  • Refactorización de procedimientos almacenados
  • Mejoras progresivas sin afectar continuidad operativa
